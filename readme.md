## 📌 About the Project

A modern responsive travel landing page built using **HTML, SCSS, and JavaScript**.  
The project focuses on building a real-world UI with interactive components, modular styling architecture, and smooth user experience.

---

## 🚀 Key Features

### 🎠 Advanced Interactive Sliders
- Built with Swiper.js
- Multiple synchronized carousels (destinations + testimonials)
- Looping, smooth transitions, and navigation controls

---

### 📱 Fully Responsive Layout
- Mobile-first design approach
- Adaptive navigation and grid restructuring
- Optimized layout across desktop, tablet, and mobile

---

### 🍔 Smart Mobile Navigation System
- Burger menu with overlay interface
- Scroll locking when menu is open
- Click-outside-to-close behavior for better UX

---

### 📍 Hero Booking Interface
- Location input with autocomplete (datalist)
- Custom date input integration
- Clean, minimal booking UI simulation

---

### 💬 Multi-Column Testimonial System
- Three-column synchronized slider layout
- Staggered scrolling effect for dynamic visual rhythm
- Repeating loop for continuous content flow

---

### 🧩 Scalable SCSS Architecture
- Modular structure using BEM methodology
- Separated base styles and reusable components
- Easy-to-maintain block-based styling system

---

### ⚡ Performance-Oriented UI
- SVG sprite icons (optimized loading)
- Lightweight image handling with responsive formats
- Clean DOM structure for fast rendering

---

### ✉️ Newsletter Component
- Simple email subscription UI
- Clean icon-enhanced input design
- Focus on conversion-style layout
